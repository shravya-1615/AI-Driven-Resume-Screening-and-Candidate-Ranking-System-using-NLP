# 🤖 AI-Driven Resume Screening and Candidate Ranking System

An AI-powered resume screening system that automatically analyzes resumes and ranks candidates based on their similarity with a given job description using Natural Language Processing (NLP) and Machine Learning techniques.

This system helps HR teams and recruiters quickly identify the best candidates by automating the resume evaluation process.

# 🚀 Features

✅ Upload multiple resumes (PDF)

✅ Enter job description

✅ Automatic resume analysis using NLP

✅ Candidate ranking based on similarity score

✅ Best candidate identification

✅ Interactive visualization dashboard

✅ Simple and user-friendly GUI using Gradio

# 🧠 Technologies Used

🐍 Python	Core programming language

🧠 Natural Language Processing (NLP)	Resume text analysis

📊 TF-IDF Vectorization	Feature extraction

📏 Cosine Similarity	Resume matching

🎛 Gradio	GUI dashboard

📈 Matplotlib	Data visualization
